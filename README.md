# IDOFactory

White-label Decentralized IDO Launchpad Platform

## Installation

1. Install dependencies

    ```bash
    yarn
    ```

2. Start app

    ```bash
    yarn start
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
